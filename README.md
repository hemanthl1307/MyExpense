MyExpense

A simple daily expense tracker that runs in your browser. Log what you spend, see daily and monthly totals, and view your spending on a chart — no install, no server, no sign-up.

Files
index.html — the page structure
style.css — all the styling
Chart.js is loaded from a CDN link inside index.html, so you need an internet connection the first time you open it (browsers cache it after that)
