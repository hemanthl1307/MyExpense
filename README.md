# 💰 MyExpense

A simple daily expense tracker that runs in your browser.

## Features

- Add daily expenses
- View daily and monthly totals
- Track spending with a chart
- No sign-up or installation required

## Files

- `index.html` – Page structure
- `style.css` – Styling
- `script.js` – Expense tracking logic
- **Chart.js** – Loaded via CDN in `index.html`
